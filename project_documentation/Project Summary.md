Project Summary – Retail Sales Analytics
Objective

To analyze retail sales data and uncover insights related to regional performance, product categories, customer segments, and seasonal trends.

Approach

Excel Analysis

Used pivot tables to explore sales and profit trends

Identified initial patterns in regions and categories

Data Cleaning (Python)

Processed raw data using Pandas

Created additional features for analysis

Dashboard Development (Power BI)

Built an interactive dashboard

Developed key performance metrics using DAX

Key Findings

West region is the strongest market in terms of sales and profit

Technology products generate the highest profit

Furniture has poor profitability despite high sales

Consumer segment contributes the majority of revenue

Sales peak during November and December

Business Impact

The analysis provides actionable insights that can help businesses:

Improve pricing strategies

Optimize product focus

Plan for seasonal demand

Allocate resources effectively

Data cleaning and transformation
